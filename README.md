🍽️ Restaurant Website

A modern, responsive restaurant website built using **HTML5** and **CSS3**. This project features a clean design with sections for the menu, about, gallery, and contact information.

## 📌 Features

- Responsive design

- Modern navigation bar

- Hero section

- About Us section

- Menu section

- Gallery

- Contact form

- Footer with social media links

- Clean and professional UI

## 🛠️ Technologies Used

- HTML5

- CSS3

## 📁 Project Structure

```

restaurant-website/

│── index.html

│── style.css

│── README.md

└── images/

    ├── hero.jpg

    ├── about.jpg

    ├── pizza.jpg

    ├── burger.jpg

    ├── pasta.jpg

    └── dessert.jpg

```

## 🚀 Getting Started

1. Download or clone the project.

2. Open the project folder in Visual Studio Code.

3. Open `index.html` in your web browser.

## 🎨 Customization

You can:

- Change colors in `style.css`

- Replace images in the `images` folder

- Add new menu items

- Update contact information
